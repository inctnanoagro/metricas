feat: Implementar sistema completo de parsing de produções Lattes

Implementação robusta de sistema de parsing para extrair produções
científicas de arquivos HTML exportados da Plataforma Lattes.

## Principais funcionalidades

- Parser router com registry extensível
- 4 parsers específicos (artigos, capítulos, jornais)
- Parser genérico como fallback
- Schema JSON canônico v2.0.0
- Suite completa de testes (165 testes passando)
- Validação contra JSON Schema
- Garantia de determinismo
- Fingerprints SHA1 para deduplicação

## Cobertura

- 27 fixtures processados com sucesso (100%)
- 3137 items extraídos
- Performance: ~540 items/segundo
